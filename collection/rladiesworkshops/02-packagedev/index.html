The Basics of R Package Development

R packages are fundamental to coding in R. In this workshop, we covered the basics of R packages using R Studio and devtools. We also covered best practices for writing R functions, and creative uses for R packages. This workshop took place on April 21st and 28th as part of the UW R-Ladies Group.
